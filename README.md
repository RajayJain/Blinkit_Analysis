## 📌 Project Overview

This project focuses on analyzing Blinkit's sales performance, customer satisfaction, and inventory distribution using Python-based data analytics techniques. The analysis uncovers valuable business insights that help identify sales trends, customer preferences, outlet performance, and optimization opportunities.

The project was developed in a Jupyter Notebook environment using powerful Python libraries for data manipulation and visualization.

💼 **The file can be viewed here [Blinkit Analysis](https://github.com/RajayJain/Blinkit_Analysis/blob/5b381d708fbecdea03d1e19d5f92722f0ae929a7/Resources/Blinkit_Analysis.ipynb)**

---

## 🎯 Business Objective

The primary objective of this analysis is to:

✅ Analyze overall sales performance

# 🚀 How to Run the Project

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/blinkit-analysis.git
```

## 2️⃣ Navigate to the Project Folder

```bash
cd blinkit-analysis
```

## 3️⃣ Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

## 4️⃣ Launch Jupyter Notebook

```bash
jupyter notebook
```

## 5️⃣ Open the Notebook

Run:

```bash
Blinkit_Analysis.ipynb
```
